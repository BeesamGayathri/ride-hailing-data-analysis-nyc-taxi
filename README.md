# 🚖 Ride-Hailing Data Analysis (NYC Taxi Dataset)

## 📌 Project Overview
This project focuses on analyzing ride-hailing data from the NYC Taxi dataset to uncover patterns in trip behavior, pricing, and demand trends. The goal is to extract meaningful insights that can support data-driven decision-making.

---

## 🎯 Objectives
- Analyze trip patterns and ride frequency
- Understand fare distribution and pricing trends
- Identify peak demand hours
- Explore relationships between trip distance and fare

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- Streamlit (for dashboard)

---

## 📊 Key Insights
- Peak demand observed during rush hours
- Strong relationship between trip distance and fare
- Certain zones show consistently high ride activity
- Pricing patterns vary based on time and location
