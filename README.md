# 🚖 Ride-Hailing Data Analysis (NYC Taxi Dataset)

## 📌 Project Overview
This project focuses on analyzing ride-hailing data from the NYC Taxi dataset to uncover patterns in trip behavior, pricing, and demand trends. The goal is to transform raw data into meaningful insights that support data-driven decision-making.

I have built this project using real-world data and created an interactive dashboard to visualize insights effectively.

---

## 🎯 Objectives
- Analyze trip patterns and ride frequency  
- Understand fare distribution and pricing trends  
- Identify peak demand hours  
- Explore relationships between trip distance and fare  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Streamlit  

---

## 📊 Key Insights
- Peak demand observed during rush hours  
- Strong correlation between trip distance and fare pricing  
- Certain locations consistently show high ride activity  
- Pricing trends vary based on time and trip distance
